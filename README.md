# Matrix library implementation
Matrix with better memory manager + exception-proof code

## How to launch

1. copying project
```
git clone git@github.com:2APetrin/matrix.git
cd matrix
```

2. making it
```
cmake -B build
cd build
make
```

3. launching
```
./src/matrix
```

4. end-to-end tests (from matrix directory)
```
./test/test.sh
```
